Backend (Node + TypeScript + Express)

Setup rápido:

1. Entre na pasta `backend`:

   - `cd backend`
2. Instale dependências:

   - `npm install`
3. Configure variáveis de ambiente em `.env` (exemplo abaixo):

   PORT=3333
   DB_HOST=localhost
   DB_USER=root
   DB_PASS=senha
   DB_NAME=tcc_formatura

4. Rodar em modo dev:

   - `npm run dev`
