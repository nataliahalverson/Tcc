Carrossel acessível — diretrizes

  • Adicionar role="region" e aria-label para identificar o carrossel.
  • Usar botões com aria-controls para anunciar slide atual e total.
  • Pausar autoplay ao hover/focus, permitir navegação por setas (← →) e tab.
  • Fornecer indicadores (dots) com estado aria-current="true" no slide ativo.

Exemplo de JS minimal para melhorar o carrossel

// melhorar: adicionar keyboard, aria, autoplay control
const slides = document.querySelectorAll('.slide');
const prev = document.querySelector('.prev');
const next = document.querySelector('.next');
let current = 0; let autoplayId = null;
function show(i){ slides.forEach((s, idx)=>{
  s.classList.toggle('active', idx===i);
  s.setAttribute('aria-hidden', idx===i? 'false':'true');
}); current = i; }
prev.addEventListener('click', ()=> show((current-1+slides.length)%slides.length));
next.addEventListener('click', ()=> show((current+1)%slides.length));
document.addEventListener('keydown', e=>{ if(e.key==='ArrowRight') next.click(); if(e.key==='ArrowLeft') prev.click(); });
